### Installation

1. Clone the repository

```bash
 git clone 'repository url'
```
2. Change directory

```bash
cd bento-grid-main

```  
3. Run npm build 

```bash
npm build

```

4. Start the development server with npm run watch

```bash
npm run watch

```
5. Open your browser and navigate to 
```bash
http://localhost:3000 to view the application.
```
